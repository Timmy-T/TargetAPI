# TargetAPI
This application is a sample application developed as part of the Target interview process.

## Application Architecture
- Python3.6
- Flask
- TinyDB
- Heroku

## Testing Instructions
The application is currently hosted on Heroku at https://flask-targetapi.herokuapp.com/


The request URL is https://flask-targetapi.herokuapp.com/products/{id}. 

GET,PUT and POST all respond to that URL with an integer ID. 

## A sample Postman collection is located in this repository to assist with testing
### Instructions for the collection
1. Download Postman https://www.getpostman.com/products
2. Import the collection called "TargetAPI.postman_collection.json"
3. Select any of the requests to run them.


Note - Sample price is already populated for Big Lebowski.
